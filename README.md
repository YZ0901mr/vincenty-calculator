# Vincenty大地测量计算器

一个基于Vincenty算法的专业大地测量计算工具。

## 功能特性

- 📍 坐标求距离：计算两点间的距离和方位角
- 🧭 距离求坐标：根据起点、方位角和距离计算终点坐标  
- 🔄 坐标转换：支持十进制度和度分秒格式转换
- 📱 响应式设计：完美适配手机和电脑

## 使用方法

1. 打开 `index.html` 文件
2. 选择需要的功能选项卡
3. 输入坐标数据
4. 点击计算按钮查看结果

## 技术说明

- 纯前端JavaScript实现
- 基于WGS-84椭球体模型
- 使用Bootstrap 5 UI框架
- 支持离线使用
